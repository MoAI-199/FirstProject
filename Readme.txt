内容物
　・Framewark2020フォルダ
　　　開発環境。solutionフォルダと同じ階層にないと.slnでデバッグができません。
　　
　・solutionフォルダ
　　　ソリューションを内包したフォルダ
　
　・Other
　　　開発に使用したプログラム以外のファイル

　・package
　　　実行して実際のゲームをプレイすることができる.exeファイルとそのresourceがあります。


　・DiscerningCode
	こだわりのあるコードをコメント共に記載

  ・Portfolio.pdf
	作品を見る際はこちらのPDFを見てください。
	作品紹介やアピールポイントが記載されています。	


動作確認済み環境
　・OS
　	Windows10
　・開発環境
	Microsoft Visual Studio 2019

操作方法
　・基本的にUIはマウスクリックで選択できます。
　・ボールの移動：マウス
　・AorDボタン　：選択移動
　・Wボタン	：決定
　・Sボタン	：決定されている線があるなら消す。
